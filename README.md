## 💻 Getting started

### Requirements

**Clone the project and access the folder**

```bash
$ gh repo clone victorserpa/ToDoList
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm install

# Start the client
$ npm run dev
```
